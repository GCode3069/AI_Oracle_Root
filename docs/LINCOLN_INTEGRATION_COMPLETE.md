## üêõ Troubleshooting

### Common Issues

**"No API keys configured"**
- System works without keys (uses free fallbacks)
- For premium results, set keys:
```powershell
$env:ELEVENLABS_API_KEY = "key"
$env:RUNWAYML_API_SECRET = "key"
$env:ANTHROPIC_API_KEY = "key"
```

**"ElevenLabs API Error"**
- Check quota at elevenlabs.io dashboard
- Verify API key is correct
- System auto-falls back to gTTS

**"RunwayML Generation Failed"**
- Check credits at runwayml.com
- System auto-falls back to MoviePy
- Still produces functional videos

**"No videos generated"**
- Check terminal output for errors
- Verify Python packages installed
- Run setup: `.\Invoke-LincolnHorror.ps1 -SetupOnly`
