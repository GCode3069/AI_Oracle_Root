## **TROUBLESHOOTING** ğŸ”§

### **Desktop Generator Won't Launch**
**Symptom:** Double-clicking does nothing
**Solution:**
```powershell
# Check Python installation
python --version

# If Python not found, install Python 3.8+
# Download from: https://www.python.org/downloads/

# Install required packages
pip install requests beautifulsoup4 lxml numpy scipy pillow
```

### **GUI Opens But Freezes**
**Symptom:** Interface appears but doesn't respond
**Solution:**
- Close and relaunch
- Check if another instance is running
- Restart computer if needed

### **"No module named 'tkinter'"**
**Symptom:** Error about tkinter missing
**Solution:**
```powershell
# Reinstall Python with tkinter support
# Or install tkinter separately:
# Windows: Should be included in Python
# Linux: sudo apt-get install python3-tk
```

### **Videos Generate But Have No Sound**
**Symptom:** Silent videos
**Solution:**
- Check ElevenLabs API key in script
- Verify internet connection
- Check audio file in `audio/` folder
- Test with command-line generator

### **FFmpeg Errors**
**Symptom:** "FFmpeg not found" or video creation fails
**Solution:**
```powershell
# Check FFmpeg installation
ffmpeg -version

# If not installed:
# Download from: https://ffmpeg.org/download.html
# Add to PATH environment variable
```
